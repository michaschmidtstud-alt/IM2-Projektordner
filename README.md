# Sonnenauf- und Untergangszeiten auf SAC-Hütten im Berner Oberland

Dieses Projekt entstand im Rahmen eines Studienprojektes der FHGR im Studiengang Multimedia Production.

Funktion:

Es lassen sich SAC-Hütten auswählen, ein Datum und dann wird die Sonnenauf- und Untergangszeiten angezeigt. <br>
Zusätzlich werden die wichtigsten Hütteninfos zu der ausgewählten Hütte angezeigt. <br><br>

Bei den Hütten haben wir uns auf SAC-Hütten im Berner Oberland beschränkt, welche alle durch einen T3 Wanderweg erreichbar sind.
